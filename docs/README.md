# Heed documentation

The main [README](../README.md) is the full overview and covers everything
briefly. These pages go deeper on each tool.

- [The studio (browser UI)](studio.md). Record, train, evaluate, and export from
  your browser with `heed ui`. The fastest way to make a model.
- [Export and deploy](export-and-deploy.md). The ONNX and TFLite formats, the
  `wake.json` preprocessing contract, and how to run a model in Python, the
  browser, mobile, and native code.
- [Train in Colab](colab.md). The zero-install notebook trainer on a free GPU.
- [Self-host with Docker](docker.md). Run the studio in a container.
- [Mobile (iOS and Android)](mobile.md). The React Native demo.
- [Browser and JavaScript](browser-and-js.md). The client-side reference.

CLI help is always one command away: `heed --help`, or `heed <command> --help`.
